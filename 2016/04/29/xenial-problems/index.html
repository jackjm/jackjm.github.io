###Ubuntu 16.04 LTS xenial problems

Just tasted the recently released Ubuntu 16.04 LTS xenial, came across some problems.

  1. The default python on Ubuntu 16.04 LTS xenial is v3.5, and there’s no python v2.7.
    If you want to install Python2, a “python-minimal” package is recommended by the system.
    It is a minimal subset of the python language, lacks of some essential modules.

For instance, no “pipes” module, the module is an interface to shell pipelines.
This “pipes” module is required on managed node for ansible.

So you have to install the full python2 on Ubuntu 16.04.

sudo apt install python*
